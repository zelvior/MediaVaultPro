# MediaVaultPro

Welcome to **MediaVaultPro**, a sophisticated media management web application designed for effortless organization and tracking of your digital collections, including movies, series, games, and more.

## 🌐 Live Demo
Explore the live demo at: [MediaVaultPro](https://zelvior.github.io/MediaVaultPro)

---

## 🛠️ Features

### User-Friendly Interface
- Intuitive dashboard for managing your collections.
- Modal-based forms for adding and editing media entries.

### Key Functionalities
- **Global Search:** Find media titles or notes with instant results.
- **Sorting Options:** Sort based on recent additions, ratings, or alphabetical order.
- **Interactive Stats Dashboard:**
  - Displays total titles, active media, completed items, and your watchlist.
- **Status Tracking:** Tag media as *Plan to Watch*, *In Progress*, or *Completed.*
- **Rating System:** Rate your titles with easy-to-use star ratings.
- **Progress Tracking:** Visualize ongoing activities with progress bars for titles in progress.

### Import/Export
- Export your vault data as JSON for backups and offline storage.
- Quickly import your data to MediaVaultPro from JSON files to sync previous collections.

## 🚀 Installation & Usage

### Prerequisites
Ensure you have a modern browser since MediaVaultPro is optimized for:
- Responsive web design
- Cutting-edge technologies like Tailwind CSS

### Steps to Get Started
1. Clone the repository to your local development environment:
   ```bash
   git clone https://github.com/zelvior/MediaVaultPro.git
   ```
2. Open the `index.html` file in your browser to launch the app.

3. Save media entries directly in the browser's `Local Storage` for continuity each time you open the app.
